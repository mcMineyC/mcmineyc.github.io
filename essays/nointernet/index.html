How My Life Would Change If The Internet Ceased To Exist

  My life would be very different if the internet ceased to exist.  Since I self-teach myself to code, I use the internet a lot.  Almost every day, I look something up on either Stack Overflow, W3Schools, Github, or the Mozilla Developer Network.  While I could use books from the library, the books my library has are a bit out of date.  I'm not faulting the library, as most people probably wouldn't be checking out books like Sam's Teach Yourself Java in 21 Days or C++ For Dummies bi-weekly like me.  If I bought the latest version of all the books that equated to the knowledge stored on just the Mozilla Developer Network, it would be very expensive.  Another way my life would change without the internet is that I wouldn't be able to learn how to make 3D animations using tutorials from channels like Grant Abbitt and CG Geek.  Not only are videos helpful when doing complex modeling, but they are a necessity in knowing what buttons to press in which order.  However, if the entire internet was going to go down, there would probably be signs leading up to it.  Certain aspects of websites not working, some websites down entirely, constant errors, that sort of thing.  In such a situation, I would make a complete offline copy of the websites I use the most frequently.  But if a hacker, say, installed malware on every single server on the entire internet to completely wipe all data when triggered, then yes, the internet would cease to exist and there wouldn't be time to make copies of the websites.  In that case, I wouldn't be able to make a copy of the websites I use the most.  Yes, I get that it was a theoretical question, but that's what I would do if it actually happened.
